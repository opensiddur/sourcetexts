Open Siddur Source Texts
===========

The [sources](./sources/) directory contains original sources that were either directly committed here or were downloaded using the  import preprocessors in the [opensiddur-ai](https://github.com/opensiddur/opensiddur-ai) repository. These texts are intended to be converted to Jewish Liturgy TEI using the Open Siddur importers and stored as [projects](https://github.com/opensiddur/opensiddur-projects/)
